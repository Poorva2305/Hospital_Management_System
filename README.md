# Hospital_Management_System
A Hospital Management System where you can:
•	Manage patient records (add, view, delete patients)
•	Manage doctors (add, view doctor profiles)
•	Book appointments between patients and doctors
•	View dashboard with statistics
Technology Stack:
•	Frontend: HTML, CSS, JavaScript (what users see)
•	Backend: Python Flask (handles requests)
•	Database: MySQL (stores data)
